# spatiotemporalai
A Short Course on Spatiotemporal Deep Learning
The material covers:

🌐 Time series, spatial time series, and forecasting

🔗 Graph-based representations of correlated time series

🧠 Graph Neural Networks and message passing

📈 Spatiotemporal Graph Neural Networks

🌪 Handling extreme events with E-STGCN for air quality forecasting

🦠 Probabilistic epidemic forecasting with spatiotemporal engression

🚦 Dynamic STGNNs for traffic flow forecasting

🐍 Python implementation with open data and code

The goal is simple: to help phd researchers and practitioners understand how graph deep learning can model complex dependencies across locations and time, while also connecting the mathematics to real-world implementation. The material includes theory, model formulation, applications, and hands-on Python practice.
